Nightmare

Made from a Unity 3d tutorial (https://unity3d.com/learn/tutorials/projects/survival-shooter-project), this game follows a kid who falls asleep and experiences a surreal zombie apocalypse in his dreams.
Survive an onslaught of zombies for as long as you can.
Installation instructions:

  1. Download the zip file found at the provided link.
  2. Make sure that Builds.exe and Builds_Data folder are in the same directory before playing.

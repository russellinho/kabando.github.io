Client/Server Platformer

This is a simple multiplayer platformer game that uses the Processing networking API to connect multiple clients to a server using localhost as the IP address. This means that the server and client must be on the same machine.
There is no objective here; it's just a simple platformer that was created with networking capabilities. I also implemented replay functionality. Directions on how to use replays is found below or in the README file.

A = Begin recording a replay A = Stop recording a replay (pressed a second time after the first) W = Run the replay at normal speed (pressed before starting the replay) E = Run the replay at 2x speed (pressed before starting the replay) R = Run the replay at 0.5x speed (pressed before starting the replay) Q = Begin the replay (will only work after you've completely recorded a replay by pressing A twice)

Installation instructions:

  1. Download the zip folder at the "Download here" link below.
  2. Extract the contents from the zip folder to a directory on your PC.
  3. To start the game, double click on the MultithreadedServer.jar file.
  4. To connect clients to the server, double click on MultithreadedClient.jar ONLY while MultithreadedServer.jar is already running.
